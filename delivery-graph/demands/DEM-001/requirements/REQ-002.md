# REQ-002

Demand: DEM-001
Priority: should

## Statement

O sistema deve permitir recuperação de senha via fluxo esqueci-minha-senha baseado em token de uso único e expiração, entregue por e-mail.

## Acceptance

- Solicitar reset gera token de uso único com expiração e não revela se o e-mail existe
- Token válido permite definir nova senha; token usado/expirado é rejeitado
- Nova senha é gravada como hash bcrypt e invalida o token

## Validation

Method: automated-test

Required evidence:

- Teste pytest do fluxo de reset: geração
- expiração
- uso único e troca efetiva de senha
