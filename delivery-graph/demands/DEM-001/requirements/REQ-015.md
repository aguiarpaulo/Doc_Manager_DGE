# REQ-015

Demand: DEM-001
Priority: must

## Statement

O sistema deve ser implantável via Docker Compose subindo API, PostgreSQL e MinIO de forma integrada.

## Acceptance

- docker compose up sobe API
- PostgreSQL e MinIO conectados
- API alcança banco e MinIO usando configuração por variáveis de ambiente
- Configuração sensível (segredos/credenciais) vem de env
- não hardcoded

## Validation

Method: manual-review

Required evidence:

- Verificação de que o compose sobe os serviços e o health check passa (smoke)
