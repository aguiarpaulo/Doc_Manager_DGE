# REQ-016

Demand: DEM-001
Priority: must

## Statement

O sistema deve garantir durabilidade: backup diário do banco PostgreSQL e redundância dos arquivos do MinIO via volumes Docker persistentes.

## Acceptance

- Existe rotina de backup diário do PostgreSQL (ex.: pg_dump agendado) documentada e executável
- Arquivos do MinIO residem em volume persistente que sobrevive a recriação do container
- Procedimento de restauração do backup está documentado e testável

## Validation

Method: manual-review

Required evidence:

- Roteiro/teste verificando geração de backup e persistência de dados após recriar containers
