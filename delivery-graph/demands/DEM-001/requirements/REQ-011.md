# REQ-011

Demand: DEM-001
Priority: must

## Statement

O sistema deve registrar histórico/auditoria imutável de toda ação relevante — login, upload, download, aprovação, rejeição e exclusão — com autor, ação, alvo e timestamp.

## Acceptance

- Cada ação relevante gera um registro de auditoria com autor
- ação
- alvo e timestamp
- Histórico de um documento pode ser consultado em ordem cronológica
- Registros de auditoria não podem ser editados/apagados via API

## Validation

Method: automated-test

Required evidence:

- Teste pytest verificando emissão de eventos de auditoria para cada tipo de ação
