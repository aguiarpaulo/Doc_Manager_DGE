# REQ-017

Demand: DEM-001
Priority: should

## Statement

O sistema deve incluir uma UI minima em Streamlit que permita login e exercitar os fluxos principais (listar/filtrar documentos, upload, aprovar/rejeitar, ver historico) consumindo a API.

## Acceptance

- Usuario faz login pela UI e recebe sessao autenticada contra a API
- UI lista documentos respeitando o escopo do usuario e permite upload/aprovacao/busca

## Validation

Method: manual-review

Required evidence:

- Verificacao manual (smoke) dos fluxos principais na UI Streamlit contra a API
