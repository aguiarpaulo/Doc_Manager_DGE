# REQ-004

Demand: DEM-001
Priority: must

## Statement

O sistema deve gerir usuários com um dos papéis Administrador, Diretor, Engenheiro ou Financeiro; apenas Administrador cria/edita/desativa usuários.

## Acceptance

- Admin cria usuário com papel; não-admin recebe 403 ao tentar
- Papel inválido é rejeitado na validação
- Usuário desativado não consegue autenticar

## Validation

Method: automated-test

Required evidence:

- Teste pytest de CRUD de usuários com checagem de papel e 403 para não-admin
